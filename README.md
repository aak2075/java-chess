# 기능 요구 사항

---

## 체스판 초기화

- [ ] 8x8 크기의 보드 생성
- [ ] 피스 위치 초기화
    - [ ] 검정색 피스 초기화
    - [ ] 흰색 피스 초기화
- [ ] Piece 추상 클래스
    - [ ] 인스턴스로 색을 가짐
    - [ ] 움직인다
    - [ ] 퀸
    - [ ] 킹
    - [ ] 룩
    - [ ] 비숍
    - [ ] 나이트
    - [ ] 폰
    - [X] 빈칸
        - [X] 메서드를 호출하면 예외 발생
- [ ] Square
    - [ ] 인스턴스로 Piece를 가짐
- [ ] Rank
    - [ ] Square의 일급 컬렉션
- [ ] Board
    - [ ] Rank의 일급 컬렉션

## 출력

- [ ] 게임 시작을 입력받는다
- [ ] 게임 종료를 입력받는다
- [ ] 보드 출력
    - [ ] 검은색 피스는 대문자로 출력
    - [ ] 하얀색 피스는 소문자로 출력
